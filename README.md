# IP-UIE
This is a Tensorflow implement of IP-UIE.
The original structure of our code is based on Retinex-Net(https://github.com/weichen582/RetinexNet).

### Requirements ###
1. Tensorflow >= 1.12.0
2. python 3.6
3. Numpy
4. pillow

### Testing ###
```shell
python main.py --phase=test
```

### Training ###
```shell
python main.py --phase=train
```
